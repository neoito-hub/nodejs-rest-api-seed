## HTTP status codes

Ripped from [here](https://restfulapi.net/http-status-codes/) and [here](https://www.restapitutorial.com/httpstatuscodes.html)

```
200 - for GET
201 - for POST
204 - for DELETE, PUT

400 - bad payload/request
401 - requires auth
403 - not authorized to do stuff
404 - not found duh!

500 - server b0rked
```
